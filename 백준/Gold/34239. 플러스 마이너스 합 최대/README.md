### 문제 설명

<p>길이 $N$의 수열 $A$에 대해 함수 $f(l,r)$은</p>

<p>$$f(l,r) =\sum_{i = l}^{r} (-1)^{\,i - l} A_i$$</p>

<p>로 정의된다.</p>

<p>$(1 \le l \le r \le N)$을 만족하는 모든 정수 순서쌍 $(l,r)$에 대하여 $f(l,r)$의 최댓값을 구하여라.</p>

### 입력 

 <p>첫째 줄에 $N$이 주어진다. $(1 \le N \le 200\,000)$</p>

<p>둘째 줄에 $N$개의 정수 $A_1, A_2, \ldots, A_N$이 공백으로 구분되어 주어진다. $(-10^9 \le A_i \le 10^9)$</p>

### 출력 

 <p>첫째 줄에 $f(l,r)$의 최댓값을 출력한다.</p>

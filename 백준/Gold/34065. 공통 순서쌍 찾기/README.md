### 문제 설명

<p>$1$부터 $N$까지의 수가 정확히 한 번씩 등장하는 수열 $A$와 $B$가 주어진다. 아래 조건을 만족하는 두 정수의 순서쌍 $(x,y)$를 $K$개 찾으시오.</p>

<ul>
	<li>$1\leq x,y\leq N$</li>
	<li>수열 $A$에서 $x$가 $y$보다 먼저 등장한다.</li>
	<li>수열 $B$에서 $x$가 $y$보다 먼저 등장한다.</li>
</ul>

### 입력 

 <p>첫째 줄에 정수 $N$과 $K$가 공백으로 구분되어 주어진다.</p>

<p>둘째 줄에 수열 $A$의 원소 $A_1, A_2, \cdots, A_N$이 공백으로 구분되어 주어진다.</p>

<p>셋째 줄에 수열 $B$의 원소 $B_1, B_2, \cdots, B_N$이 공백으로 구분되어 주어진다.</p>

<p>수열 $A$와 $B$는 각각 $1$부터 $N$까지의 정수가 정확히 한 번씩 등장하는 길이가 $N$인 수열이다.</p>

### 출력 

 <p>조건을 만족하는 순서쌍이 $K$개 이상 있다면 첫째 줄에 <code>Yes</code>를 출력하고, 이후 $K$개의 줄에 걸쳐 순서쌍 $(x, y)$의 원소를 한 줄에 한 쌍씩 공백으로 구분하여 출력한다.</p>

<p>조건을 만족하는 순서쌍이 $K$개 미만이라면 <code>No</code>를 출력한다.</p>

<p>만약 가능한 답이 여러 개 있다면, 그중 하나를 아무 것이나 출력해도 정답으로 인정된다.</p>

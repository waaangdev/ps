### 문제 설명

<p>$1$부터 $N$까지의 수가 하나씩 임의로 배열된 길이 $N$의 수열 $A_1, A_2,..., A_N$이 주어진다. 주어진 수열에서 연속된 $K$개의 수를 고른 후, 선택한 $K$개의 수의 구간 내에서 $K$개의 수를 자유롭게 재배열하는 연산을 한 번 수행한다.</p>

<p>연산을 수행한 후, 수열에서 만들 수 있는 연속된 증가하는 부분 수열의 최대 길이를 구하시오.</p>

<p>증가하는 수열이란, 모든 $i < j$에 대해 $A_i < A_j$인 수열을 말한다.</p>

### 입력 

 <p>첫 번째 줄에 정수 $N$, $K$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 정수 $A_1, A_2, ..., A_N$이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>첫 번째 줄에 수열에서 만들 수 있는 연속된 증가하는 부분 수열의 최대 길이를 출력한다.</p>

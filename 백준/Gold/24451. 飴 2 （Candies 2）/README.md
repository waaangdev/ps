### 문제 설명

<p>机の上に <var>N</var> 個の飴が横一列に並んでおり，左から順に <var>1</var> から <var>N</var> までの番号が付けられている．飴 <var>i</var> (<var>1 ≦ i ≦ N</var>) の美味しさは <var>A<sub>i</sub></var> である．</p>

<p>JOI 君は，<var>N</var> 個の飴のうちいくつかを選んで食べることにした．</p>

<p>ただし，飴を食べ過ぎないために，どの連続する <var>K</var> 個の飴についても，そのうち高々 <var>2</var> 個しか食べないようにする．すなわち，どの <var>j</var> (<var>1 ≦ j ≦ N - K + 1</var>) についても，飴 <var>j</var> から飴 <var>j + K - 1</var> までの連続する <var>K</var> 個の飴のうち，食べる飴の個数は <var>2</var> 個以下でなければならない．</p>

<p>このもとで，JOI 君は食べる飴の美味しさの合計をできるだけ大きくしたい．</p>

<p><var>N</var> 個の飴の美味しさと <var>K</var> が与えられたとき，JOI 君が食べる飴の美味しさの合計の最大値を求めるプログラムを作成せよ．</p>

### 입력 

 <p>入力は以下の形式で標準入力から与えられる．</p>

<pre><var>N</var> <var>K</var>
<var>A<sub>1</sub></var> <var>A<sub>2</sub></var> <var>…</var> <var>A<sub>N</sub></var></pre>

### 출력 

 <p>標準出力に，JOI 君が食べる飴の美味しさの合計の最大値を <var>1</var> 行で出力せよ．</p>

### 문제 설명

<p>정수 $N$, $K$가 주어졌을 때, 다음 조건을 만족하는 길이가 $N$인 수열 $A$를 구해보자.</p>

<ul>
<li>$1 \le i < j \le N$을 만족하는 $(i, j)$에 대해 $\text{gcd}(A_i, A_j) = 1$을 만족하는 개수가 정확히 $K$개다.</li>
</ul>

<p>모든 입력에 대해 수열을 항상 만들 수 있음이 보장된다.</p>

### 입력 

 <p>첫 번째 줄에 정수 $N, K$가 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>첫 번째 줄에 수열 $A_1, A_2, \cdots, A_N$을 공백으로 구분하여 출력한다. 가능한 수열이 여러 개라면 그중 아무것이나 출력한다.</p>

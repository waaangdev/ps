### 문제 설명

<p>어떤 수열을 오름차순으로 정렬했을 때 등차수열이 된다면 그 수열을 <strong>특별한 수열</strong>이라고 한다. 길이가 $1$인 수열은 특별한 수열이다.</p>

<p>길이가 $N$인 수열 $A$가 주어진다. 모든 $1\leq i\leq N$에 대해, $A$의 첫 $i$개의 수로 이루어진 수열이 특별한 수열인지 알아내라.</p>

### 입력 

 <p>첫째 줄에 수열의 길이 $N$이 주어진다. ($1 \leq N \leq 200\,000$)</p>

<p>둘째 줄에 수열의 원소 $A_1, A_2, \ldots, A_N$이 공백으로 구분되어 주어진다. ($1 \leq A_i \leq 10^9$)</p>

### 출력 

 <p>$N$개의 줄에 걸쳐, $i$번째 줄에는 $A$의 첫 $i$개의 수로 이루어진 수열이 특별한 수열이라면 <code>1</code>을, 아니라면 <code>0</code>을 출력한다.</p>

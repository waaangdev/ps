### 문제 설명

<p>두 양의 정수 $A$, $B$가 주어질 때, 다음과 같은 조건을 만족하는 양의 정수 $C$의 개수를 $f(A,B)$라 하자.</p>

<p style="text-align: center;">$GCD(A,B) = GCD(A,C)$, $LCM(A,B) = LCM(B,C)$</p>

<p>$\sum_{i=1}^N \sum_{j=1}^N f(i,j)$의 값을 구하시오.</p>

<p>$GCD(A,B)$는 $A$와 $B$의 최대공약수를, $LCM(A,B)$는 $A$와 $B$의 최소공배수를 의미한다.</p>

### 입력 

 <p>양의 정수 $N$이 주어진다. $(1 \leq N \leq 10^9)$</p>

### 출력 

 <p>$\sum_{i=1}^N \sum_{j=1}^N f(i,j)$의 값을 출력한다. 문제의 입력 범위 안에서 이 값은 $2^{63}$ 미만임이 보장된다.</p>
